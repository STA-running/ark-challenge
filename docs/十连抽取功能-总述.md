# 十连抽取功能 · 完整规则总述

> 版本：v4.0 设计定稿
> 日期：2026-06-02

---

## 1. 核心一句话

> **编队阶段可选「十连抽取」替代手动选干员，与手动模式互斥。**

---

## 2. 操作流程总览

```
关卡+标签确认
    ↓
编队阶段 → 选择 [✋手动] 或 [🎰十连]
    ↓
十连模式开始：
  ┌─────────────────────────────────────┐
  │ 第 1 轮：抽 10 人 → [不要/锁定]      │
  │ 第 2 轮：抽 10 人 → [不要/锁定]      │
  │ 第 3 轮：抽 10 人 → [不要/锁定]      │
  │                                     │
  │ 每轮「不要」上限 10 次               │
  │ 锁定后 10 人从合法池永久移除          │
  └─────────────────────────────────────┘
    ↓ 3 轮锁定后
┌──────────────────────────────────────┐
│ 从锁定池（最多 30 人）+ 通用干员编队    │
│ 只能在锁定范围中选择                   │
└──────────────────────────────────────┘
```

---

## 3. 核心规则

| 规则 | 值 |
|------|-----|
| 与手动模式 | **互斥** |
| 每次抽取人数 | **10 人**（不放回） |
| 可锁定轮数 | **3 轮** |
| 每轮「不要」上限 | **10 次**（用完后必须锁定） |
| 通用干员 | 1★2★ **无需抽取**，可直接选入编队 |
| 最终编队来源 | **仅限**锁定池 3 批 + 通用干员 |
| 标签约束 | **全程生效**（合法池由标签过滤） |

---

## 4. 概率规则

### 第 1~9 抽

| 星级 | 概率 |
|------|------|
| 6★ | 10% |
| 5★ | 20% |
| 4★ | 60% |
| 3★ | 10% |

### 第 10 抽

先掷一枚硬币判断是否出 6★，未命中则走常规加权：

| 星级 | 概率规则 | 有效概率计算（表达式 → 值） |
|------|---------|------------------------|
| 6★ | 先判 50% 命中 → 出 6★；剩余 50% 按 10% 加权 | **50% + (50% × 10%) = 55%** |
| 5★ | 未命中时按 50% × 20% 加权 | **50% × 20% = 10%** |
| 4★ | 未命中时按 50% × 60% 加权 | **50% × 60% = 30%** |
| 3★ | 未命中时按 50% × 10% 加权 | **50% × 10% = 5%** |
| **合计** | | **55% + 10% + 30% + 5% = 100%** |

> 仅合法池中有 6★ 干员时触发 50% 判定，否则跳过直接走常规加权。

### 其他

- **6★ 上不封顶**：一轮十连可能出多个 6★
- **无 6★ 池**：合法池中无 6★ 时，第 10 抽按常规概率
- **不放回**：同一次十连不重复抽到同名干员

---

## 5. 合法池计算

```
合法池 = 所有干员
       - 被当前已确认标签禁用的
       - 已被锁定轮次包含的
```

十连只从合法池中抽取。锁定后该批干员移出合法池。

## 6. 通用干员框

```
📋 通用干员 · 1★2★        （不消耗抽卡次数）
┌─────┬─────┬─────┐
│ 夜刀│ 巡林│ 黑角│ ← 点击即加入编队
│ ★★ │ ★★ │ ★★ │
└─────┴─────┴─────┘
┌─────┬─────┬─────┐
│ 12F │ 杜林│ 芬  │
│ ★   │ ★   │ ★★ │
└─────┴─────┴─────┘
```

- 免抽：直接点击加入编队
- 受标签约束：不合规的干员灰显
- 可与十连锁定干员混编

## 8. 边界情况

| 场景 | 行为 |
|------|------|
| 合法池 < 10 人 | 能抽多少抽多少 |
| 合法池 = 0 | 「十连」按钮禁用 |
| 抽不满 3 轮（池不够） | 提前结束 |
| 「不要」用完（10 次） | 按钮禁用，仅剩「锁定」可选 |
| 编队满 12 人 | 停止选取 |
| 切换标签后 | 锁定池和已编干员不变 |

## 9. 可配置常量（代码中集中定义）

```typescript
export const PULL_CONSTANTS = {
  PULLS_PER_BATCH: 10,           // 每次抽取人数
  MAX_BATCHES: 3,                 // 最多锁定轮数
  MAX_REROLLS_PER_BATCH: 10,      // 每轮重抽上限

  BASE_RATES: {                   // 第 1~9 抽概率
    6: 0.10, 5: 0.20, 4: 0.60, 3: 0.10,
  },
  TENTH_PULL_SIXSTAR_RATE: 0.50,  // 第 10 抽的 6★ 概率
};
```

改了这些常量，十连的连数/轮数/概率就全变了。
