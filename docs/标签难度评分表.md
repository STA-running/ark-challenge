# 标签难度评分表 — 人工填写

> 评分范围：1（简单）~ 10（极难），请根据标签对通关难度的影响打分。
> 分数用于计算本局挑战的总体难度，不参与游戏逻辑。

---

## 评分标准

| 分数 | 含义 | 示例 |
|------|------|------|
| 1~2 | 轻微影响 | 总费≤120、编队≤8 |
| 3~4 | 中等限制 | 同职≤2、单费≤25 |
| 5~6 | 显著难度 | 无医疗、仅近战位 |
| 7~8 | 极具挑战 | 编队≤4、零阻挡 |
| 9~10 | 极限生存 | 无6★+零阻挡+编队≤4 组合 |
