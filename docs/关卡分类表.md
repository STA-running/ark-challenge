# 关卡分类表 — 自限挑战规则器

> **用途**：请在此表中为每个关卡填写"类型"（普通/Boss/超难/超难Boss），标注完后我会据此生成 `stages.json`。
>
> **收录规则（已执行）**：
> - ✗ 不收录序章及第1~4章（太简单）
> - ✗ 不收录S关（支线）
> - ✓ 第5章起收录
>
> **分类标准**：
> - **普通关**：无Boss、无特殊高难机制的常规战斗
> - **Boss关**：含命名Boss的关卡
> - **超难关**：非Boss但难度极高（H关默认归此类，除非含Boss则归超难Boss）
> - **超难Boss关**：同时满足Boss+超难
>
> **难度模式说明**：
> - 所有关卡默认包含"标准"和"突袭"两种难度，无需标记
> - 突袭 = 磨难 = 险地（不同关卡名称不同，等价）
> - **沙盘推演**仅第15/16/17章的**非超难关**拥有，已自动标记 ✓
> - H关均为超难/超难Boss，**无沙盘推演**
