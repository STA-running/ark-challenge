基于明日方舟官网 (ak.hypergryph.com) 和鹰角网络官网 (hypergryph.com) 的设计语言分析，
输出多层CSS背景系统：
- 底色 `#131518`（暖灰深色基调）
- 40px 战术网格（灰线 opacity 0.16）

- **body 背景**：7层CSS渐变叠加（CRT扫描线 → 暗角 → 蓝紫辉光 → 青绿辉光 → 中央聚光 → 40px战术网格 → 纯黑基底）
