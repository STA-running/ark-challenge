# 2026-06-04 工作日志

## 修复：焰狐龙梓兰干员头像不显示

**根因**：SquadPage.tsx 中 `PROFESSION_ICONS` 的 8 个职业图标 URL 全部 404（PRTS Wiki 文件结构变更），导致：
1. 任何头像加载失败后 fallback 到职业图标也 404 → 无限 error 循环 → 裂图
2. 所有直接使用职业图标作为默认图标的干员也受影响

**修复内容**：
- `src/pages/SquadPage.tsx`：将 PROFESSION_ICONS 从远程 PNG URL 替换为内联 SVG data URI（`makeProfIcon()` 函数），彻底消除外部依赖
- `src/pages/SquadPage.tsx`：3 个 `onError` handler 增加 `dataset.fallback` 防无限循环保护
